Default methods

Back to one of examples:


transactions.sort(comparing(Transaction::getValue));

List::sort is new method in Java 8. Previously you used Collections::sort

Default methods


interface List {
  default void sort(Comparator<? super E> c){
    Collections.sort(this, c);
  }
}

Basically default methods are methods that can be put also inside interfaces.

limitations

You can use only what's availabe from this interface:
  1. parameters
  2. declared methods
  3. global static methods
  4. constants
